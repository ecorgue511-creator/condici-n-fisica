Protección de datos

Protección de datos

En esta Situación de Aprendizaje utilizaremos aplicaciones móviles para registrar la actividad física, por lo que es fundamental hacer un uso responsable de las mismas.

El alumnado deberá proteger su información personal, datos de salud y ubicación, evitando compartirlos públicamente o sin supervisión. Asimismo, se fomentará el uso de configuraciones de privacidad adecuadas y la descarga de aplicaciones seguras.

Además, se trabajará el respeto a la privacidad de los demás, evitando difundir datos o imágenes sin consentimiento.

👉 Para facilitar la comprensión de estos aspectos, el alumnado podrá consultar la siguiente infografía elaborada específicamente para esta SdA:
